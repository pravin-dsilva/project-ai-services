# Limitations

The following section outlines the current constraints and known issues of the tool, based on its present design and implementation.

- User authentication and multi-tenancy are not supported.
- Image Processing in PDFs is not supported.
- The tool supports only a single LPAR environment; multi-LPAR configurations are not currently supported.
- The tool is supported only on RHEL operating systems.
- Only PDFs ingestions and processing is supported.
- UI and catalog features are planned for future but are not included in the current scope.
